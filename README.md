# دفتر التاجر

تطبيق إدارة الحسابات والمعاملات التجارية مصمم للمحال التجارية الصغيرة والمتوسطة.

## المميزات

- إدارة الحسابات النقدية والبنكية
- تتبع الديون والمستحقات
- إدارة العملاء والموردين
- نظام النسخ الاحتياطية السحابية
- تقارير مالية شاملة
